 نجوم دلتا – دليل النشر السريع على Netlify
═══════════════════════════════════════════════════════════════

هذه الحزمة هي بناء جاهز (production build). فقط ارفعها كما هي.

──────────────────────────────────────────────
الطريقة 1 — السحب والإفلات (الأسرع)
──────────────────────────────────────────────
1) اذهب إلى:  https://app.netlify.com/drop
2) اسحب محتويات هذا المجلد (وليس المجلد نفسه) إلى المنطقة المخصصة.
3) سيتم النشر تلقائياً خلال 30-60 ثانية.

──────────────────────────────────────────────
الطريقة 2 — عبر سطر الأوامر (CLI)
──────────────────────────────────────────────
   npm install -g netlify-cli
   netlify deploy --dir=. --functions=netlify/functions --prod

──────────────────────────────────────────────
متغيّرات البيئة المطلوبة (Site settings → Environment variables)
──────────────────────────────────────────────
المطلوبة لتشغيل OTP والدفع:

  AUTHENTICA_API_SECRET   = (من https://app.authentica.sa)
  MOYASAR_SECRET_KEY      = (من https://dashboard.moyasar.com)
  MOYASAR_WEBHOOK_SECRET  = (من Moyasar webhooks)

اختيارية (تُحقن داخل البناء — إذا أردت تغييرها يجب إعادة البناء):
  VITE_SUPABASE_URL
  VITE_SUPABASE_ANON_KEY
  VITE_FIREBASE_API_KEY
  VITE_GEMINI_API_KEY
  VITE_GOOGLE_MAPS_API_KEY
  VITE_MOYASAR_PUBLIC_KEY

──────────────────────────────────────────────
بعد النشر
──────────────────────────────────────────────
1) اربط النطاق المخصص: deltastars.store
2) فعّل HTTPS تلقائياً (Netlify يفعّله بالنقرة الواحدة).
3) جرّب: https://your-site.netlify.app
   - يجب أن تظهر شاشة الترحيب ثم المتجر خلال 2-3 ثوانٍ.
   - إذا علقت شاشة البداية > 12 ثانية، يظهر زر "إعادة تحميل المتجر".

──────────────────────────────────────────────
استكشاف الأخطاء
──────────────────────────────────────────────
• شاشة بيضاء: نظّف cache المتصفح (Ctrl+Shift+R)
• OTP لا يصل: تأكد من AUTHENTICA_API_SECRET في متغيرات Netlify
• دفع لا يعمل: تأكد من MOYASAR_SECRET_KEY + ربط webhook
